# LootRarity
LootRarity

LootRarity adds new bless tiers that can appear on any equippable drop. Each tier boosts stats and is color-coded (icons sparkle, tooltips match). Vanilla items still occur—these tiers are rare bonuses.

Tiers & Bonuses

Uncommon (Green, +4): +20% to item stats

Rare (Yellow, +5): +30%

Epic (Red, +6): +40%

Legendary (Orange, +7): +50%

Mythic (Obsidian/Light Black, +8): +70%

Drop Chances (per eligible item)

Uncommon (+4): 5.0%

Rare (+5): 2.5%

Epic (+6): 1.0%

Legendary (+7): 0.6%

Mythic (+8): 0.01%

Highlights

Works on any equippable loot from enemies.

Sparkles and name color match the tier.

Scaling is applied directly via the game’s stat calcs (no fake numbers).
